# Amazon Transcribe

* 자동 음성 인식(ASR, Automatic Speech Recognition) 서비스이다.
* 입력은 오디오이고, 출력은 텍스트이다.
* 이 기능을 강화하는 다양한 기능을 제공한다.

  * 언어 커스터마이징
  * 프라이버시 보호용 필터
  * 대상 청중에 맞는 언어 조정
  * 화자 식별
* 또한 사용 사례에 맞게 사용자 지정 어휘(Custom Vocabulary)와 언어 모델을 구성할 수 있다.
* 과금은 전사한 오디오 길이를 기준으로 초당 사용량 기반으로 이루어진다.
* 제품은 여러 형태로 제공된다.

  * Transcribe
  * Transcribe Medical
  * Transcribe Call Analytics
* Amazon Transcribe의 사용 사례

  * 검색이 가능하도록 오디오를 전체 텍스트로 인덱싱
  * 회의록 작성
  * 자막, 캡션, 전사문 생성
  * Call Analytics를 사용해 통화 데이터를 수집하고, 특성 분석, 요약, 카테고리 분류, 감성 분석 수행
* API를 통해 애플리케이션과 통합할 수 있으며, 다른 AWS ML 서비스와도 연동된다.
