# AWS Audit Manager

* **AWS Audit Manager**는 AWS 사용 현황을 **지속적으로 감사(continuous audit)** 하도록 도와, **리스크/컴플라이언스(규제·표준 준수) 관리**를 단순화하는 서비스입니다.
* **사전 구축(Prebuilt) 프레임워크**를 제공하며, 각 프레임워크는 특정 **규제/표준(예: ISO, SOC, PCI 등)** 에 맞춰 **평가(assessment)를 구조화**하고 **증거 수집을 자동화**합니다.
* 원하는 프레임워크를 기반으로 **Assessment(평가)** 를 생성할 수 있습니다.
* Audit Manager는 **자동 리소스 평가(resource assessments)** 를 수행하여, **감사 범위(in-scope)** 로 지정한 **AWS 계정들**에서 필요한 데이터를 수집하고 **Evidence(증적)** 로 축적합니다.
* 감사가 끝나 더 이상 증적 수집이 필요 없으면 **Evidence collection(증거 수집)을 중지**할 수 있습니다.

**키워드 한줄:** *Audit Manager = 컴플라이언스 “평가 + 증적 자동 수집”을 프레임워크 기반으로 운영하는 서비스*
